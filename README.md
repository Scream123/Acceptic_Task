#веб-приложение для регистрации и аутентификации пользователей

### Настройка перед запуском проекта

1. Проект тестировался на параметрах: PHP 5.6 | Apache-2.4 | MYSQL-5.5
2. Скопируйте корневую папку в папку домена вашего локального сервера или же хоста
3. Измените адрес страницы, если это потребуется, в файле `config.php`
4. Импортируйте  файл БД в свою СУБД


#### Запуск проекта
- Введите в адресной строке 'http://accepticnew.loc или же 'свой_url'
